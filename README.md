# DemoGraphicVisualization

Demographic data from Eurostat RestAPI service visualized with DevExtreme and D3 UI Controls.

  

### Technologies
* .NET Core 3.1
* LINQ
* Vue.js 2
* Axios
* D3
* DevExtreme 20.2

### Software environment
* Node.js 12.6.3
* .NET Core SDK 3.1.408
